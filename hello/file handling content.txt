open() ---> file = open("filename.format","mode)
mode: r(read), w(write), a(append), x(create), b(binary), t(text)
file.close() ---> close the file
file.read() ---> read the file
file.write() ---> write the file
file.append() ---> append the file
file.seek() ---> seek the file
file.tell() ---> tell the file
file.readline() ---> readline the file



r -> read the file which is existed
w -> write the data in file
fileno() -> file number
flush() -> fle stream refresh
